## Prereqs

- npm
- docker (with docker-compose)

## Development

To start development server:

```bash
npm run dev
```

This will create a database container. To clean up:

```bash
npm run postdev
```

## Test

Run all tests:

```bash
npm run test
```

Run only unit test:

```bash
npm run test:unit
```

Run only integration test:

```bash
npm run test:integration
```
